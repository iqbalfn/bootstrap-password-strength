Password Strength

Use Bootstrap's JavaScript password strength plugin to update some progress element about strength quality of an input password value.

The build-in function is just a simple password strength tester. You can implement your own tester if you have one.

Examples

Default

Try to input some weak password and see the progress element.

Usage

Via JavaScript

Call an element with selector with single line of JavaScript.

$('#element').pwdstr()

Options

Options can be passed via data attribute or JavaScript. For data attributes, append the option name to data-, as in data-progress="...".

Name Type Default Description
progress string
element		 null The selector of the progress element to update.
tester string
function		 Build-in The function to use to test the strength of the password. This function is called with the first parater is the callback function that expect to call with final test value, the second parameter is element value and the third parameter is the input element it self.
